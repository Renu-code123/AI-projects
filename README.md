# ⚙️ Installation & Setup

## 🔽 Clone the Repository

```bash
git clone https://github.com/Renu-code123/AI-projects.git
```

## 📁 Move to Project Folder

```bash
cd AI-projects
```

## 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Run Jupyter Notebook

```bash
jupyter notebook
```
